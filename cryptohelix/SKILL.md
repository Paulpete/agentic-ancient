# CryptoHelix: Blockchain Mastery

Multi-chain intelligence engine for Empire Infinity Matrix. Query multiple blockchains simultaneously, analyze portfolios, track yields, and optimize cross-chain strategies.

## Quick Start

### Portfolio Analysis
```bash
scripts/chain_intelligence.py portfolio <wallet-address> solana ethereum polygon
```

Shows total value, chain distribution, and top holdings.

### Transaction History
```bash
scripts/chain_intelligence.py transactions <address> ethereum
```

Recent transactions with status and gas.

### Token Search
```bash
scripts/chain_intelligence.py search USDC
```

Find tokens across all chains.

### Relayer Switching
```bash
scripts/chain_intelligence.py relayer solana helius
```

Switch between providers for optimal performance.

## Supported Chains

- **Solana**: Via Helius, Quicknode, Solscan
- **Ethereum**: Via Etherscan, Alchemy, Moralis
- **Polygon**: Via Polygonscan, Alchemy
- **Arbitrum**: Via Arbiscan, Alchemy
- **NEAR**: Via NEAR API
- **Bitcoin**: Via blockchain explorers

## Features

- **Multi-Chain Portfolio**: Aggregate balances across all chains
- **Real-time Prices**: USD valuations for all assets
- **Transaction Tracking**: Complete history with gas optimization
- **Relayer Orchestration**: Auto-failover between providers
- **Smart Caching**: Reduce API calls, improve speed

## Integration

Works seamlessly with Helix Nexus for orchestrated workflows.
